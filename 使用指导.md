## 通过DevBox快速共建包
这里通过tree.hnp作为示例
### 1、开发者准备好鸿蒙化的hnp包后，将编出的hnp文件放入项目中`hnp/arm64-v8a`目录下

### 2、修改`entry/src/main/module.json5`文件，将hnp包信息加入列表。
```
      {
        "package": "tree.hnp",
        "type": "public"
      }
```
### 3、添加二进制使用命令手册`entry/src/main/ets/pages/manual_data.ets`
```
{
  name: 'tree',
  description: '以树状结构列出目录内容，显示文件和目录的层次关系。',
  syntax: 'tree [-acdfghilnpqrstuvxACDFJQNSUX] [-L level [-R]] [-H [-]baseHREF] [-T title] [-o filename] [-P pattern] [-I pattern] [--gitignore] [--gitfile[=]file] [--matchdirs] [--metafirst] [--ignore-case] [--nolinks] [--hintro[=]file] [--houtro[=]file] [--inodes] [--device] [--sort[=]name] [--dirsfirst] [--filesfirst] [--filelimit[=]#] [--si] [--du] [--prune] [--charset[=]X] [--timefmt[=]format] [--fromfile] [--fromtabfile] [--fflinks] [--info] [--infofile[=]file] [--noreport] [--hyperlink] [--scheme[=]schema] [--authority[=]host] [--opt-toggle] [--version] [--help] [--] [directory ...]',
  options: [
    { paramName: '-a', description: 'All files are listed.' },
    { paramName: '-d', description: 'List directories only.' },
    { paramName: '-l', description: 'Follow symbolic links like directories.' },
    { paramName: '-f', description: 'Print the full path prefix for each file.' },
    { paramName: '-x', description: 'Stay on current filesystem only.' },
    { paramName: '-L level', description: 'Descend only level directories deep.' },
    { paramName: '-R', description: 'Rerun tree when max dir level reached.' },
    { paramName: '-P pattern', description: 'List only those files that match the pattern given.' },
    ...... // 篇幅太长省略
  ],
  examples: [
    'tree',
    'tree -L 2',
    'tree -a -h -p',
    'tree -I "node_modules|.git"',
    'tree -o output.txt /path/to/directory',
    'tree --help'
  ],
  showDetails: false
}
```
### 4、添加涉及的开源软件声明`entry/src/main/ets/about_diaglog/opensource_list.ets`
```
  {
    "name": "tree",
    "license": LicenseType.GPL_V2L,
    "source": "https://github.com/hongtaoh/tree-1.8.0",
    "licenseFile": "about_tree_opensource.html",
    "showLicense": false
  },
```
### 5、添加对应的开源声明html文件`entry/src/main/resources/rawfile`
about_`xxx`_opensource.html  如`about_tree_opensource.html`
### 6、运行devbox项目，在终端HiShell中测试命令是否可用（安装后终端需重新打开）
``` 
tree
```
